📝 វិញ្ញាសាប្រឡងតេស្តសមត្ថភាព

មេរៀន៖ វ៉ិចទ័រទីតាំង និងការចែកអង្កត់ក្នុងលំហ
សំណួរទី ១៖ គេមានវ៉ិចទ័រទីតាំង a = (3, -1) និង b = (7, 5)។ ចូររកវ៉ិចទ័រទីតាំងនៃចំណុចកណ្តាល I នៃអង្កត់ AB។
សំណួរទី ២៖ ចំណុច C ចែកអង្កត់ AB ខាងក្នុងតាមផលធៀប m : n = 2 : 1។ បើ a និង b ជាវ៉ិចទ័រទីតាំងនៃ A និង B តើវ៉ិចទ័រទីតាំង c ស្មើនឹងប៉ុន្មាន?
សំណួរទី ៣៖ គេឲ្យចំណុច A(1, 2) និង B(4, 5)។ វ៉ិចទ័រទីតាំង d នៃចំណុច D ដែលចែកអង្កត់ AB ខាងក្រៅតាមផលធៀប 2 : 1 គឺ៖
សំណួរទី ៤៖ បើចំណុច P(5, y) ជាចំណុចកណ្តាលនៃអង្កត់តភ្ជាប់ពី A(x, 4) ទៅ B(3, 8) ចូររកតម្លៃ x និង y។
